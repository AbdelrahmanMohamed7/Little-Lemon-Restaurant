A modern React application for the Little Lemon Mediterranean Restaurant in Chicago.

## Features
* **Table Reservation:** Fully functional booking form with validation.
* **Responsive Design:** Clean, semantic HTML5 and CSS Grid layout.
* **Navigation:** Smooth routing between Home, About, and Booking pages.

## Technologies Used
* **React** (Hooks, Router)
* **CSS Grid & Flexbox**
* **JavaScript** (Unit testing with Jest/Vitest)

## How to Run
1. Clone the repo.
2. Run `npm install`.
3. Run `npm start`.
